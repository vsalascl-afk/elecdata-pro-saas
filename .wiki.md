# Project Summary
**ElecData Pro** is a **multi-tenant web application** developed for **Comtel S.p.A.** that streamlines the management of **Work Orders (OT)** across various companies. The platform empowers organizations to effectively manage their work orders with features such as user authentication, comprehensive work order creation with photo uploads, digital signature capture, and real-time status tracking. The **Admin Portal** has been enhanced to include user management features like password resets and now allows superadmins to delete users efficiently, improving overall management capabilities.

# Project Module Description
The application includes the following core functionalities:
1. **Login/Authentication**: Secure user sign-in via Supabase Auth.
2. **Admin Panel**: Manage company accounts, including user creation, role assignment, password resets, and user deletions.
3. **Create Work Order**: Form for entering work order details, with photo uploads and technician assignment.
4. **Edit Work Order**: Modify existing work orders in "pending" or "in progress" states, with validation for completion.
5. **KPIs Display**: Dashboard showcasing key performance indicators for work orders.
6. **Doughnut and Bar Charts**: Visualize work order statuses and trends using Chart.js and Recharts.
7. **Work Order List**: View details of work orders with status changing options, photo viewing, and PDF export functionality.
8. **State Filtering**: Filter work orders by current status and technician name.
9. **Sidebar Menu**: Responsive navigation for orders, dashboard, admin panel, and company management.
10. **User Menu**: Displays user avatar, initials, and name for better identification.
11. **Multi-Tenant Management**: Super admins can manage multiple companies, including profile and branding configurations.
12. **Completion Validation**: Work orders require at least 2 photos and a digital signature for completion.
13. **Password Reset**: Superadmins can reset user passwords via a secure dialog.
14. **User Deletion**: Superadmins can delete users from the system while ensuring compliance with security policies.

# File Description Inventory
- **elecdata_pro_SAAS_13.html**: Main HTML file for the Work Orders application.
- **src/components/**: UI components for various functionalities, including updated validation and new features.
- **src/lib/**: Utility files such as the Supabase client and PDF export utilities.
- **src/pages/**: Main entry point for the application.
- **MANUAL_INSTALACION.md**: Installation manual for deploying the application on client domains.
- **PRESENTACION_CLIENTE.md**: Document for client presentations, providing a structured demo guide.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (authentication, PostgreSQL database, file storage)
- **Charting**: Recharts for data visualization
- **Signature Capture**: SignaturePad for digital signatures
- **PDF Generation**: jsPDF for creating PDF documents

# Usage
To get started with the project:
1. Install necessary dependencies using your package manager.
2. Build the project using the appropriate build command.
3. Run the application to start using the Work Orders system.
4. For deployment on client domains, refer to the `MANUAL_INSTALACION.md` for detailed instructions.
5. To enable the password reset feature and user deletion, add the **Service Role Key** to your environment variables in a `.env` file:
   ```
   VITE_SUPABASE_SERVICE_KEY=your_service_role_key_here
   ```
